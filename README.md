SNUwagon backend

## Local setting

Check you have downloaded `.coveralls.yml`, `init_db.sh`, `set_env.sh`

```
# execute only for the first time
./init_db.sh

# execute on each login
source ./set_env.sh
```

## Run

```
python manage.py runserver
```

## Testing (CI / coverage)

```
./ci.sh
```
